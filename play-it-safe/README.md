# Play It Safe: Manage Security Risks

This folder contains projects and activities completed as part of the
"Play It Safe: Manage Security Risks" module from the Google Cybersecurity
Professional Certificate.

The main project in this module is an internal security audit conducted
for the fictional company Botium Toys.

This repository holds a security audit performed for the fictional company
Botium Toys as part of the Google Cybersecurity Professional Certificate.

## Objective
The objective of this security audit is to evaluate the security posture of
Botium Toys, determine risks, review existing controls, and provide
recommendations for improvement to safeguard business operations and
customer data.

## Scope
The scope of this security audit includes:
- Internal IT infrastructure
- Customer data handling
- Access control processes
- Operational security processes

## Key Learning Outcomes
From this security audit, I have learned:
- How to determine the scope of the audit and assumptions
- How to determine and evaluate security risks
- How to review administrative, technical, and physical controls
- How to prepare findings and provide recommendations

## Frameworks & Standards Referenced
- NIST Cybersecurity Framework (CSF)
- PCI DSS
- General Data Protection Regulation (GDPR)
- SOC Type 1 and Type 2

## Disclaimer
This audit is for educational purposes only and does not represent a real
organization or real security assessment.
